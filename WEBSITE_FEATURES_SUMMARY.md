# Website Features Summary

This document summarizes the main features available in the AI Finance Tracker website.

## 1. Public Website

### Landing Page
- Modern marketing-style homepage introducing the platform.
- Feature highlights for financial tracking, AI assistance, and analytics.
- Quick links to sign in, register, and explore the app.
- Responsive design with animated visuals and theme-aware styling.

### About Page
- Dedicated page explaining the purpose of the project.
- Presents the product vision, core value, and app capabilities.

## 2. Authentication and Access

- User registration with profile details such as name, email, phone, gender, and password.
- User login with protected routes for authenticated areas.
- Session persistence using token-based authentication.
- Automatic redirect to the login page if the session expires.

## 3. Dashboard

- High-level monthly financial overview.
- Displays income, expenses, savings, and recent activity.
- Visual trend tracking compared with previous periods.
- Analytics widgets and charts for spending behavior.
- Quick actions for adding transactions and scanning receipts.

### AI Dashboard Insights
- Generates AI-powered financial summaries.
- Streams progress while trying available AI models in real time.
- Explains spending patterns, savings rate, and actionable recommendations.
- Supports follow-up finance questions through AI chat.

## 4. Transactions Management

- Add income and expense transactions.
- Edit existing transactions.
- Delete single transactions or bulk delete multiple records.
- Filter transactions by date range, category, and type.
- Search and browse transaction history with a polished interface.

### Category Management
- Built-in and custom categories.
- Create personal categories.
- Assign icons or emojis to categories.
- Delete custom categories when no longer needed.

## 5. Budgets

- Create category-based monthly budgets.
- Edit and delete budgets.
- Copy last month’s budgets into the current month.
- Compare actual spending against budget targets.
- Track overspending and remaining budget visually.

## 6. Savings Page

- Dedicated savings dashboard showing cash savings and total wealth.
- Tracks monthly savings goal and monthly saved amount.
- Supports long-term savings goals with target amount and date.
- Shows recent savings activity including deposits and withdrawals.
- Displays progress indicators and financial health context.

### Savings Actions
- Add deposits into savings.
- Record withdrawals from savings.
- Use a guided flow for savings-related transactions.
- Refresh live data and market-linked values when needed.

### Savings AI
- AI-generated savings analysis with actionable guidance.
- Real-time streaming progress while the AI model is working.
- Fallback summary support if live streaming is unavailable.

## 7. Investments

- Add investment positions directly from the savings page.
- Supports metals and currencies such as gold, silver, and foreign currency holdings.
- Automatically uses current market rates when creating an investment.
- Shows portfolio value, current worth, and profit/loss.
- Allows deleting investment entries.

### Investment Analytics
- Portfolio performance indicators.
- Allocation visuals and analytics cards.
- Rate history and market indicator widgets.
- Wealth movement charts and trend summaries.

## 8. Goals Page

- Create financial goals.
- Set target amounts and target dates.
- Update goal progress over time.
- Delete completed or unwanted goals.
- Motivational visual feedback such as celebratory animations.

## 9. Receipt Upload and OCR

- Upload receipt images.
- Extract receipt text using OCR.
- Use AI to convert raw receipt data into structured transaction details.
- Detect merchant, amount, date, and likely category.
- Confirm and edit parsed data before saving the transaction.

## 10. Shopping and Pantry Management

- Maintain shopping and inventory state.
- Save pantry items and shopping list items.
- Track local shopping-related planning data.
- Analyze shopping and pantry information using AI.
- Uses real-time or fallback AI chat behavior for shopping insights.

## 11. Reports

- Generate financial reports for a selected date range.
- Supports downloadable report generation.
- Helps summarize activity outside the live dashboard view.

## 12. Profile Management

- View current user information.
- Update editable profile fields.
- Profile-aware layout with user avatar support.
- Keeps user identity visible across the app shell.

## 13. Global AI Chat Widget

- Floating chat widget available across the website.
- Lets users ask finance-related questions from any page.
- Provides a quick-access assistant without leaving the current screen.

## 14. Design and User Experience

- Responsive interface for desktop and smaller screens.
- Dark and light theme support.
- Sidebar navigation for all core app areas.
- Toast notifications, modals, animations, and polished transitions.
- Rich charting and dashboard-style cards for financial clarity.

## 15. Core Backend Capabilities Supporting the Website

- Authenticated API for all major finance workflows.
- Transactions, categories, budgets, goals, savings, and investment endpoints.
- AI endpoints for dashboard summaries, chat, and savings analysis.
- Receipt OCR endpoints for upload and confirmation.
- Shopping state persistence endpoints.
- Report generation endpoint.
- Health and root endpoints for service status.
